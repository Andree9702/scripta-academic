# Carpeta de Clientes — SCRIPTA ACADEMIC

## Estructura por cliente

Para cada cliente nuevo, crear una subcarpeta:

```
07_CLIENTES/
└── [APELLIDO_NOMBRE]/
    ├── brief.md
    ├── contrato.md (o .pdf firmado)
    ├── cotizacion.md
    ├── notas.md
    └── entregables/
```

## Reglas

1. Cada cliente tiene su propia carpeta
2. Toda comunicación relevante se registra en notas.md
3. Los entregables finales van en la subcarpeta entregables/
4. CONFIDENCIAL — nunca compartir contenido entre clientes
