# Testimoniales

## Instrucciones para solicitar testimoniales

Enviar este mensaje a cada coautor:

"Hola [NOMBRE], estoy lanzando un servicio de producción editorial
académica llamado SCRIPTA ACADEMIC. Como hemos trabajado juntos en
[PROYECTO], me gustaría mucho contar con un breve testimonio tuyo
para mi página web. ¿Podrías escribirme 3-4 oraciones sobre tu
experiencia trabajando conmigo? Incluiría tu nombre y afiliación.
¡Gracias!"
